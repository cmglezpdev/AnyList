# Dev

1. Copy and rename the ```.env.template``` file to ```.env```.
2. Fill the environment variables.
3. Install project depentencies.

```bash
yarn install
```

4. Up postres image in docker

```bash
docker-compose up -d
```

5. Up App

```bash
yarn start:dev
```

6. Go to site ```localhost:3000/graphql```
